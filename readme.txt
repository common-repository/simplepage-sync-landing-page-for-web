== Description ==

SimplePage – Sync Landing Page For Web. Đồng bộ các dự án Landing Page đã thiết kế tại nền tảng tạo Landing Page miễn phí SimplePage.vn lên Website Wordpress của bạn.


== Installation ==

1. From your WordPress Dashboard go to 'Plugins' > 'Add new' and search for 'SimplePage – Sync Landing Page For Web'.
2. Click 'Install Now' under the title of the SimplePage – Sync Landing Page For Web plugin.
3. Click activate.
4. Go to 'Settings' > 'SimplePage – Sync Landing Page For Web' and insert key Landing Page in https://simplepage.vn/dashboard/user-profile
5. Choose Landing Page and choose Page in your wordpress
6. Click 'Save' and you're done!
